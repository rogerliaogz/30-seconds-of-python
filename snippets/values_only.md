Returns a flat list of all the values in a flat dictionary.

- Use `dict.values()` to return the values in the given dictionary.

```py
def values_only(flat_dict):
  return list(flat_dict.values())
```

```py
ages = {
  'Peter': 10,
  'Isabel': 11,
  'Anna': 9,
}
values_only(ages) # [10, 11, 9]
```
